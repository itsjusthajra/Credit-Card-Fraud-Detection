# Credit-Card-Fraud-Detection

Built a credit card fraud detection system using Logistic Regression and Random Forest on a highly imbalanced dataset, performing end-to-end EDA, feature scaling, and stratified data splitting. Applied SMOTE to improve fraud detection recall to 92% with Logistic Regression, and developed a class-weighted Random Forest model achieving strong performance (precision: 0.81, recall: 0.82, ROC-AUC: 0.98). Evaluated models using confusion matrix, classification metrics, and ROC-AUC, and identified key predictors through feature importance analysis.
